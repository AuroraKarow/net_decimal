高精度十进制数。这是一个为 AI 测试与实验的高精度十进制数计算库。运算库包括了三个部分，分别是：

- [函数](wiki/function.md)
- [字段](wiki/field.md)
- [运算符](wiki/operator.md)

函数包括了最基本的运算函数，例如对数、三角函数和幂函数等。通过访问字段属性以及修改字段配置参数，可以更改计算的精度，获取数字的绝对值等属性。运算符包括了基础运算符和位运算符。
